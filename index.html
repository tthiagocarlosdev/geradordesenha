Gerar Senha

Este sistema foi desenvolvido para gerar senhas seguras.

Para uma senha ser considerada segura, ela deve apresentar:

  • Mínimo de 8 caracteres;
  • Letras maiúsculas;
  • Letras minusculas;
  • Números;
  • Símbolos.

Digite a quantidade de caracteres de sua senha:

[ATENÇÃO]! Digite valores de 8 a 40.
